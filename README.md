# AveDex em Python

Projeto desenvolvido para praticar Python, Git e GitHub por meio de um menu interativo sobre aves.

## Conteúdos praticados

* `print()` e `input()`
* Variáveis
* `if`, `elif` e `else`
* `while`
* Manipulação de strings com `.strip()`
* Git: `init`, `add`, `commit` e `push`
* Publicação de repositório no GitHub

## Como executar

```bash
python avedex.py
```
